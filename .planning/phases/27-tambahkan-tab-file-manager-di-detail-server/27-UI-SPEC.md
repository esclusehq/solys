## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #080b15 (dark) / #f8fafc (light) | Page background |
| Secondary (30%) | #0d0f1a (dark) / #ffffff (light) | Panel backgrounds |

## Component Structure

### Tab Integration (ServerDetailsPage.jsx)

- Existing tabs: dashboard, files, plugins, backups, settings, ide, source
- File Manager tab already present at `activeTab === 'files'`
- No new tab addition needed — FileManager component integration required

### Split View Layout

| Section | Width | Content |
|---------|-------|---------|
| Tree Navigator | 240px (resizable optional) | Collapsible folder tree, lazy-loaded |
| File List | flex-1 | Current directory contents, breadcrumbs, toolbar |

### Tree Navigator (Left Panel)

- Collapsible folder structure with expand/collapse chevrons
- Lazy loading on folder expand (per D-04)
- Click folder to navigate to that directory
- Shows folder icons (📁) and file icons
- Highlight current path

### File List (Right Panel)

- Breadcrumb navigation at top
- Toolbar with: View toggle (list/tree), New File, New Folder, Upload, Paste, Bulk actions
- Table columns: Checkbox, Name, Size, Modified, Actions
- Row hover state with action buttons revealed
- Support for drag-and-drop upload

### Existing Modals (from FileManager.jsx)

- Edit file modal (full-screen overlay)
- New folder modal (centered dialog)
- New file modal (centered dialog)
- Delete confirmation modal (centered dialog with destructive styling)
- Rename modal (centered dialog)
- Compress modal (centered dialog)

All modals use backdrop blur and glass-panel styling.
