    **A. Edit `app/index.html` — Add flash-guard inline script before `<body>` or immediately inside `<head>` (per UI-SPEC copywriting contract Example 1).**

    The script MUST be placed BEFORE `<div id="root">` and the React script `<script type="module" src="/src/main.jsx">`. It reads localStorage for `escluse-ui` → if stored theme found, sets data-theme → ELSE reads matchMedia('prefers-color-scheme: light') → sets data-theme.

    Insert this script block BEFORE line 20 (`<body>`) or immediately inside `<head>`:

    ```html
        <!-- ═══ Theme flash guard — runs before React mounts ═══ -->
        <script>
            (function() {
                try {
                    var stored = JSON.parse(localStorage.getItem('escluse-ui') || '{}');
                    if (stored && stored.state && stored.state.theme) {
                        document.documentElement.setAttribute('data-theme', stored.state.theme);
                    } else {
                        var prefersLight = window.matchMedia('(prefers-color-scheme: light)').matches;
                        document.documentElement.setAttribute('data-theme', prefersLight ? 'light' : 'dark');
                    }
                } catch(e) {
                    document.documentElement.setAttribute('data-theme', 'dark');
                }
            })();
        </script>
    ```

    Per the UI-SPEC Research: this inline script MUST run synchronously before any rendering. Place it AFTER the Google Fonts link and the analytics script, but BEFORE `</head>`. The exact insertion point is right before `</head>` at line 18.
