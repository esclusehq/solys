Add to Server impl block:

```rust
/// Returns true if server is in a running state (container_running or running)
pub fn is_running(&self) -> bool {
    matches!(self.status.as_str(), "running" | "container_running")
}

/// Returns true if server is stopped (not running, not starting, etc)
pub fn is_stopped(&self) -> bool {
    !self.is_running()
}

    continue;
}
```

This ensures only running servers are polled for status.
