  <action>
Transform install.sh into a curl-downloadable installer:

1. Add shebang detection for curl pipe:
```bash
# Detect if run via curl pipe or local execution
if [ -z "$INSTALL_SOURCE" ] && [ -z "$curl_used" ]; then
    # Set flag to indicate this script was piped via curl
    curl_used=1
fi
```

2. Add self-download capability (when run via curl, the script should be able to download itself):
```bash
# If called with --download flag, output this script to stdout
if [ "$1" = "--download" ]; then
    cat "$0"
    exit 0
fi
```

3. Add OS detection function:
```bash
detect_os() {
    if [ -f /etc/os-release ]; then
        . /etc/os-release
        echo "$ID"
    elif [ -f /etc/centos-release ]; then
        echo "centos"
    else
        echo "unknown"
    fi
}
```

4. Support distributions: ubuntu, debian, centos, fedora, rhel, almalinux

5. Add podman/docker installation:
```bash
install_dependencies() {
    local os="$1"
    case "$os" in
        ubuntu|debian)
            apt-get update && apt-get install -y curl podman || docker
            ;;
        centos|fedora|rhel|almalinux)
            dnf install -y curl podman || docker
            ;;
    esac
}
```

6. Add binary download from get.esluce.com:
```bash
download_binary() {
    local version="${1:-latest}"
    curl -sSL "https://get.esluce.com/releases/solys-${version}" -o /opt/solys/solys
    chmod +x /opt/solys/solys
}
```

7. Add interactive config prompts:
```bash
prompt_config() {
    echo "Enter your backend URL (e.g., https://api.escluse.com): "
    read -r backend_url
    echo "Enter your API key: "
    read -rs api_key
    # Replace placeholders in config.toml with actual values
    sed -i "s|PLACEHOLDER_BACKEND_URL|$backend_url|g" /etc/solys/config.toml
    sed -i "s|PLACEHOLDER_API_KEY|$api_key|g" /etc/solys/config.toml
}
```

8. Add main flow:
```bash
main() {
    echo "Detecting OS..."
    local os=$(detect_os)
    echo "Detected: $os"
    
    echo "Installing dependencies..."
    install_dependencies "$os"
    
    echo "Downloading binary..."
    download_binary "latest"
    
    echo "Configuring..."
    prompt_config
    
    echo "Starting service..."
    systemctl enable solys
    systemctl start solys
}
```

The script must work both:
- Locally: ./install.sh
- Via curl: curl -sSL https://get.esluce.com/agent | bash

This means the script content (when accessed via curl) serves as BOTH the installer AND the download target.
