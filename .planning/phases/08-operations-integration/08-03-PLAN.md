<objective>
Wire the existing agent SFTP handlers (agent/solys/src/handlers/sftp.rs) through the REST API and frontend. The agent already has `sftp.upload` and `sftp.download` task handlers that transfer files via SSH/SFTP, but they are completely disconnected from the API and frontend. This plan adds the missing protocol dispatch, API endpoints, and frontend glue.

**Purpose:** Deliver FILE-01 (SFTP file browsing — already works via existing `list_files` → agent `file.list_dir`), FILE-02 (upload via SFTP), and FILE-03 (download via SFTP). Agent-mode servers can now transfer files using SSH/SFTP transport through the agent.

**Architecture:**
1. Extend `CommandParams` (both API and agent sides) with SFTP-specific fields: `connection_key`, `local_path`, `remote_path`
2. Add `sftp_upload`/`sftp_download` command dispatch entries to `agent_connection.rs` so the agent routes incoming commands to `sftp.rs` handlers
3. Add `sftp_upload_file` and `sftp_download_file` handler functions in `file_handlers.rs` that accept files, route through the agent, and return results
4. Register routes in `server_handlers.rs`
5. Add frontend API functions in `useServers.js` and expose SFTP upload/download in `FileManager.jsx`

**Output:**
- Agent dispatches `sftp.upload` and `sftp.download` task types
- REST endpoints: `POST /:id/files/sftp-upload` and `POST /:id/files/sftp-download`
- Frontend: SFTP upload/download buttons in FileManager toolbar for agent-mode servers
</objective>

    /// Route SFTP operation through agent for agent-mode servers.
    /// Uses the sftp.upload or sftp.download task types in the agent's handler dispatch.
    async fn route_sftp_through_agent(
        container: &ApiState,
        server_id: Uuid,
        command: &str,  // "sftp_upload" or "sftp_download"
        connection_key: String,
        local_path: String,
        remote_path: String,
    ) -> Result<serde_json::Value, AppError> {
        let server = container.get_server_use_case.execute(server_id).await
            .map_err(|_| AppError::NotFound)?;
        
        if server.executor_type != "agent" {
            return Err(AppError::BadRequest("SFTP operations require agent executor type".into()));
        }
        
        let node_id = server.node_id
            .ok_or_else(|| AppError::BadRequest("Server has no node assigned".into()))?;
        
        let params = CommandParams {
            connection_key: Some(connection_key),
            local_path: Some(local_path),
            remote_path: Some(remote_path),
            ..Default::default()
        };
        
        let response = container.node_client.send_command(
            node_id,
            server_id,
            command,
            params,
        ).await.map_err(|e| AppError::InternalError(anyhow::anyhow!("Agent communication failed: {}", e)))?;
        
        if !response.success {
            return Err(AppError::InternalError(anyhow::anyhow!("SFTP operation failed: {}", response.output)));
        }
        
        // Parse the output JSON (contains status + bytes_transferred)
        let output: serde_json::Value = serde_json::from_str(&response.output)
            .map_err(|_| {
                // If not valid JSON, return the raw output as a string value
                serde_json::json!({ "status": "completed", "raw": response.output })
            });
        
        Ok(output)
    }
    ```

    /// Upload a file via SFTP through the agent.
    /// Accepts multipart form: file, connection_key, remote_path.
    /// Saves to temp location, routes through agent sftp.upload task, returns result.
    pub async fn sftp_upload_file(
        State(container): State<ApiState>,
        Path(server_id): Path<Uuid>,
        mut multipart: Multipart,
    ) -> Result<impl IntoResponse, AppError> {
        let mut file_data: Option<Vec<u8>> = None;
        let mut file_name: Option<String> = None;
        let mut connection_key: Option<String> = None;
        let mut remote_path: Option<String> = None;
        
        while let Some(mut field) = multipart.next_field().await.map_err(|e| AppError::BadRequest(e.to_string()))? {
            let name = field.name().unwrap_or("").to_string();
            match name.as_str() {
                "file" => {
                    file_name = field.file_name().map(|s| s.to_string());
                    let mut chunks = Vec::new();
                    while let Some(chunk) = field.chunk().await.map_err(|e| AppError::BadRequest(e.to_string()))? {
                        chunks.extend_from_slice(&chunk);
                    }
                    file_data = Some(chunks);
                }
                "connection_key" => {
                    let text = field.text().await.map_err(|e| AppError::BadRequest(e.to_string()))?;
                    connection_key = Some(text);
                }
                "remote_path" => {
                    let text = field.text().await.map_err(|e| AppError::BadRequest(e.to_string()))?;
                    remote_path = Some(text);
                }
                _ => {}
            }
        }
        
        let file_data = file_data.ok_or_else(|| AppError::BadRequest("Missing file field".into()))?;
        let file_name = file_name.ok_or_else(|| AppError::BadRequest("Missing filename".into()))?;
        let connection_key = connection_key.ok_or_else(|| AppError::BadRequest("Missing connection_key field".into()))?;
        let remote_path = remote_path.ok_or_else(|| AppError::BadRequest("Missing remote_path field".into()))?;
        
        // Write file to temp location
        let temp_dir = std::env::temp_dir().join("sftp-uploads").join(server_id.to_string());
        tokio::fs::create_dir_all(&temp_dir).await.map_err(|e| AppError::InternalError(anyhow::anyhow!(e)))?;
        let temp_file = temp_dir.join(&file_name);
        tokio::fs::write(&temp_file, &file_data).await.map_err(|e| AppError::InternalError(anyhow::anyhow!(e)))?;
        
        let local_path = temp_file.to_string_lossy().to_string();
        let full_remote = if remote_path.ends_with('/') {
            format!("{}{}", remote_path, file_name)
        } else {
            format!("{}/{}", remote_path, file_name)
        };
        
        let result = route_sftp_through_agent(
            &container,
            server_id,
            "sftp_upload",
            connection_key,
            local_path.clone(),
            full_remote,
        ).await?;
        
        // Clean up temp file
        let _ = tokio::fs::remove_file(&temp_file).await;
        
        Ok(Json(ApiResponse::success(result)))
    }
    ```

    /// Download a file via SFTP through the agent.
    /// Accepts JSON body: { connection_key, remote_path }.
    /// Routes through agent sftp.download task, streams result back to client.
    pub async fn sftp_download_file(
        State(container): State<ApiState>,
        Path(server_id): Path<Uuid>,
        Json(payload): Json<SftpDownloadRequest>,
    ) -> Result<impl IntoResponse, AppError> {
        let temp_dir = std::env::temp_dir().join("sftp-downloads").join(server_id.to_string());
        tokio::fs::create_dir_all(&temp_dir).await.map_err(|e| AppError::InternalError(anyhow::anyhow!(e)))?;
        
        // Extract filename from remote_path for local temp file
        let file_name = payload.remote_path
            .split('/')
            .last()
            .unwrap_or("download");
        let temp_file = temp_dir.join(file_name);
        let local_path = temp_file.to_string_lossy().to_string();
        
        let result = route_sftp_through_agent(
            &container,
            server_id,
            "sftp_download",
            payload.connection_key,
            local_path.clone(),
            payload.remote_path,
        ).await?;
        
        // Read the downloaded file
        if !temp_file.exists() {
            return Err(AppError::InternalError(anyhow::anyhow!("Downloaded file not found at {}", local_path)));
        }
        
        let file = tokio::fs::File::open(&temp_file).await.map_err(|e| AppError::InternalError(anyhow::anyhow!(e)))?;
        let stream = tokio_util::io::ReaderStream::new(file);
        let body = axum::body::Body::from_stream(stream);
        
        let response = axum::response::Response::builder()
            .status(axum::http::StatusCode::OK)
            .header(axum::http::header::CONTENT_TYPE, "application/octet-stream")
            .header(axum::http::header::CONTENT_DISPOSITION, format!("attachment; filename=\"{}\"", file_name))
            .body(body)
            .map_err(|e| AppError::InternalError(anyhow::anyhow!(e)))?;
        
        // Schedule temp file cleanup (fire and forget)
        let temp_file_clone = temp_file.clone();
        tokio::spawn(async move {
            tokio::time::sleep(std::time::Duration::from_secs(60)).await;
            let _ = tokio::fs::remove_file(&temp_file_clone).await;
        });
        
        Ok(response)
    }
    ```

                <label className="block text-xs text-[var(--color-text-muted)] mb-1">File</label>
                <input
                  type="file"
                  onChange={e => setSftpModal(prev => ({ ...prev, files: Array.from(e.target.files || []) }))}
                  className="w-full text-sm"
                />
              </div>
            )}
            
            {sftpModal.mode === 'upload' && sftpModal.files.length > 0 && (
              <div className="text-sm text-[var(--color-text-muted)]">
                Selected: {sftpModal.files.map(f => f.name).join(', ')}
              </div>
            )}
          </div>
          
          <div className="flex justify-end gap-2 mt-6">
            <button
              onClick={() => { setSftpModal(null); setSftpConnectionKey(''); setSftpRemotePath(''); }}
              className="px-4 py-2 text-sm text-[var(--color-text-muted)] hover:text-white transition-colors"
            >
              Cancel
            </button>
            <button
              disabled={sftpUploading || !sftpConnectionKey || !sftpRemotePath || (sftpModal.mode === 'upload' && sftpModal.files.length === 0)}
              onClick={async () => {
                if (sftpModal.mode === 'upload') {
                  setSftpUploading(true);
                  try {
                    for (const file of sftpModal.files) {
                      await import('../hooks/useServers').then(m => m.sftpUploadFile(serverId, file, sftpConnectionKey, sftpRemotePath));
                    }
                    setToast({ type: 'success', message: 'SFTP upload complete' });
                    setSftpModal(null);
                    loadFiles(path);
                  } catch (err) {
                    setToast({ type: 'error', message: `SFTP upload failed: ${err.message}` });
                  } finally {
                    setSftpUploading(false);
                  }
                } else {
                  setSftpUploading(true);
                  try {
                    const { sftpDownloadFile } = await import('../hooks/useServers');
                    const response = await sftpDownloadFile(serverId, sftpConnectionKey, sftpRemotePath);
                    // Create download link from blob response
                    const blob = await response.blob();
                    const url = URL.createObjectURL(blob);
                    const a = document.createElement('a');
                    a.href = url;
                    a.download = sftpRemotePath.split('/').pop() || 'download';
                    a.click();
                    URL.revokeObjectURL(url);
                    setToast({ type: 'success', message: 'SFTP download complete' });
                  } catch (err) {
                    setToast({ type: 'error', message: `SFTP download failed: ${err.message}` });
                  } finally {
                    setSftpUploading(false);
                  }
                }
              }}
              className="px-4 py-2 text-sm font-medium bg-[var(--color-cosmic-cyan)] text-[#080b15] rounded hover:opacity-90 transition-opacity disabled:opacity-50"
            >
              {sftpUploading ? 'Transferring...' : sftpModal.mode === 'upload' ? 'Upload' : 'Download'}
            </button>
          </div>
        </div>
      </div>
    )}
    ```

    The import of `useServers` functions is done dynamically inside the onClick handlers using dynamic `import()` to avoid circular dependencies and keep the existing module structure clean. If this approach conflicts with the module build system, use a static import at the top of the file instead.

    Note: The `sftpUploadFile` returns JSON, while `sftpDownloadFile` returns a blob (binary download). The download handler creates a temporary `<a>` element to trigger the browser download, which is the standard pattern for binary downloads in React.
