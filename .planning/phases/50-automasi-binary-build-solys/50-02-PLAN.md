  <action>
    Create `install.sh` at the repository root with the following exact implementation. This script MUST be a standalone, production-grade installer following the pattern established by tools like syft, hey-cli, and cosign itself.

    ```bash
    #!/usr/bin/env bash
    # Escluse Agent Installer
    # Usage: curl -fsSL https://get.esluce.com/latest/install.sh | bash
    #        curl -fsSL https://get.esluce.com/latest/install.sh | bash -s v1.2.3
    set -euo pipefail
    ```

    Key behaviors:
    1. **Version argument**: Accept optional `$1` argument (version string like `1.2.3`). Default to `latest`.
    2. **Platform detection**: Use `uname -s` for OS and `uname -m` for arch:
       - OS: `linux` stays as-is; `darwin` → `linux` (same binaries for macOS if supported; if not, exit with clear message). `mingw*|cygwin*` → `windows`.
       - ARCH: `x86_64|amd64` → `x86_64`; `aarch64|arm64` → `aarch64`; anything else → error "Unsupported architecture".
    3. **URL construction**: 
       - If version is `latest`: `BASE_URL="https://get.esluce.com/latest"`
       - If specific version (without v prefix): `BASE_URL="https://get.esluce.com/v${VERSION}"`
    4. **Archive selection by OS**:
       - Linux: `ARCHIVE="solys-linux-${ARCH}.tar.gz"`, `BINARY="escluse-agent"`, extract with `tar -xzf`
       - Windows: `ARCHIVE="solys-windows-x86_64.zip"`, `BINARY="escluse-agent.exe"`, extract with `unzip` (note: installer.sh is for Linux/macOS primarily; Windows users should use install.ps1 but handle gracefully)
    5. **Download**: Use `curl -fsSL` to download archive + SHA256SUMS.txt to temp directory (`mktemp -d`). Trap cleanup.
    6. **SHA256 verification**: `(cd "$TMPDIR" && sha256sum --ignore-missing -c SHA256SUMS.txt)`
    7. **Cosign verification** (optional, only if `cosign` in PATH): Download `SHA256SUMS.txt.bundle`, run:
       ```bash
       cosign verify-blob "$TMPDIR/SHA256SUMS.txt" \
         --bundle "$TMPDIR/SHA256SUMS.txt.bundle" \
         --certificate-identity-regexp "https://github.com/escluse/escluse/.github/workflows/release.yml@refs/tags/v" \
         --certificate-oidc-issuer "https://token.actions.githubusercontent.com"
       ```
    8. **Extract and install**: Extract archive, use `install -m 755` to copy binary to `$INSTALL_DIR`.
       - `INSTALL_DIR="${ESCLUSE_BIN_DIR:-/usr/local/bin}"`
       - `sudo install -m 755 "$TMPDIR/$BINARY" "$INSTALL_DIR/$BINARY"` (use sudo if not root)
    9. **Success message**: Print "Installed $BINARY to $INSTALL_DIR/$BINARY"
    10. **Error handling**: Exit with non-zero on any failure with descriptive message.

    The script MUST have these variables at the top:
    ```bash
    REPO="escluse/escluse"
    INSTALL_DIR="${ESCLUSE_BIN_DIR:-/usr/local/bin}"
    VERSION="${1:-latest}"
    ```

    Write the full script. Target minimum 70 lines. Ensure all error paths produce user-friendly messages. Include a `trap` for temp directory cleanup.

  <action>
    Create `install.ps1` at the repository root with the following exact implementation. This script MUST be a standalone PowerShell installer for Windows users.

    ```powershell
    #!/usr/bin/env pwsh
    # Escluse Agent Windows Installer
    # Usage: powershell -c "irm https://get.esluce.com/latest/install.ps1 | iex"
    param(
        [string]$Version = "latest"
    )
    ```

    Key behaviors:
    1. **Version parameter**: Accept optional `-Version` parameter. Default `"latest"`.
    2. **Platform detection**:
       - Architecture: `$env:PROCESSOR_ARCHITECTURE` — `AMD64` → `x86_64`, `ARM64` → `aarch64`, anything else → error.
       - OS: Assume Windows (this script is Windows-only).
    3. **URL construction**:
       - If `$Version -eq "latest"`: `$baseUrl = "https://get.esluce.com/latest"`
       - Otherwise: `$baseUrl = "https://get.esluce.com/v$Version"`
    4. **Archive selection**: `$archive = "solys-windows-${arch}.zip"`, `$binary = "escluse-agent.exe"`
    5. **Installation directory**: `$InstallDir = Join-Path $env:ProgramFiles "Escluse"`. Fallback to `Join-Path $env:LOCALAPPDATA "Escluse"` if ProgramFiles is not writable.
    6. **Download**: Use `Invoke-WebRequest -Uri "$baseUrl/$archive" -OutFile (Join-Path $tmpDir $archive)` and similarly for SHA256SUMS.txt.
    7. **Temp directory**: `Join-Path $env:TEMP "escluse-install"`. Use `New-Item -ItemType Directory -Force -Path`.
    8. **SHA256 verification**:
       ```powershell
       $expectedHash = (Get-Content (Join-Path $tmpDir "SHA256SUMS.txt") | Select-String $binary) -split '\s+' | Select-Object -First 1
       $actualHash = (Get-FileHash (Join-Path $tmpDir $archive) -Algorithm SHA256).Hash.ToLower()
       if ($actualHash -ne $expectedHash) { throw "Checksum mismatch for $archive" }
       ```
    9. **Cosign verification** (optional): Check if `cosign.exe` is in PATH. If so, download bundle and run `cosign verify-blob` (same parameters as install.sh). Wrap in try/catch — cosign verification is best-effort on Windows.
    10. **Extract**: `Expand-Archive -Path (Join-Path $tmpDir $archive) -DestinationPath $tmpDir -Force`
    11. **Install**: Create InstallDir if not exists. `Copy-Item (Join-Path $tmpDir $binary) (Join-Path $InstallDir $binary) -Force`
    12. **PATH update**: Add InstallDir to User PATH if not already present:
        ```powershell
        $userPath = [Environment]::GetEnvironmentVariable("Path", "User")
        if ($userPath -notlike "*$InstallDir*") {
            [Environment]::SetEnvironmentVariable("Path", "$userPath;$InstallDir", "User")
        }
        ```
    13. **Success message**: `Write-Host "Installed $binary to $InstallDir"`
    14. **Error handling**: `$ErrorActionPreference = "Stop"` at top. Use try/catch for download failures, checksum mismatches, and extraction errors.

    Write the full script. Target minimum 70 lines. Use PowerShell conventions:
    - `Write-Host` for user messages
    - `Write-Verbose` for verbose output
    - `throw` for fatal errors
    - Cmdlet-style naming (`Get-`, `Invoke-`, `Join-Path`, `New-Item`, `Copy-Item`)
    - Use `-Force` flag on `New-Item` to create intermediate directories
  </action>
