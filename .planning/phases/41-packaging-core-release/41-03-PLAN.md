```markdown
# Devnode Agent

A lightweight agent for managing game servers on your infrastructure.

## Quick Start

1. Extract the release package
2. Edit config.toml and set your backend_url and api_key
3. Run: sudo ./install.sh
4. Check status: systemctl status devnode-agent

## Requirements

- Linux (x86_64)
- systemd
- Root/sudo access for installation

## Configuration

Edit /etc/devnode-agent/config.toml:
- backend_url: Your API endpoint (e.g., https://api.escluse.com)
- api_key: Your authentication key

## Commands

- Install: sudo ./install.sh
- Uninstall: sudo ./uninstall.sh
- Check status: systemctl status devnode-agent
- View logs: journalctl -u devnode-agent -f
- Restart: sudo systemctl restart devnode-agent

## Troubleshooting

Check logs: journalctl -u devnode-agent -n 100
Verify config: cat /etc/devnode-agent/config.toml
```

Create the systemd service file:

```ini
[Unit]
Description=Devnode Agent - Game Server Management Agent
Documentation=https://escluse.com/docs
After=network-online.target
Wants=network-online.target

[Service]
Type=simple
User=root
WorkingDirectory=/opt/devnode-agent
ExecStart=/opt/devnode-agent/devnode-agent
Restart=always
RestartSec=10

# Environment
Environment=RUST_LOG=info

# Security hardening
NoNewPrivileges=true
PrivateTmp=true
ProtectSystem=strict
ReadWritePaths=/var/log/escluse-agent /etc/devnode-agent

[Install]
WantedBy=multi-user.target
```
