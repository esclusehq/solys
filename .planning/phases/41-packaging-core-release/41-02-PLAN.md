<task type="auto">
  <name>Task 2: Create install.sh script</name>
<files>release/package/install.sh</files>
  <action>
Create install.sh that performs full installation:

1. Check running as root
2. Create directories: /opt/devnode-agent, /etc/devnode-agent
3. Copy binary to /opt/devnode-agent/
4. Copy config.toml to /etc/devnode-agent/
5. Copy systemd service file to /etc/systemd/system/
6. Reload systemd daemon
7. Enable and start the service
8. Show success message

Make it executable: chmod +x install.sh

Include error handling and rollback on failure.
</action>
  <verify>
    <automated>test -f release/package/install.sh && test -x release/package/install.sh</automated>
  </verify>
  <done>install.sh script created with full installation logic</done>
</task>

<task type="auto">
  <name>Task 3: Create uninstall.sh script</name>
<files>release/package/uninstall.sh</files>
  <action>
Create uninstall.sh that performs full uninstallation:

1. Check if service is running, stop if so
2. Disable service
3. Remove binary from /opt/devnode-agent/
4. Remove config from /etc/devnode-agent/
5. Remove systemd service file
6. Reload systemd daemon
7. Show success message

Make it executable: chmod +x uninstall.sh

Include cleanup of any remaining files.
</action>
  <verify>
    <automated>test -f release/package/uninstall.sh && test -x release/package/uninstall.sh</automated>
  </verify>
  <done>uninstall.sh script created with full uninstallation logic</done>
